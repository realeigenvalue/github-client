# github-client

An Android app GitHub client using Android SDK API 25.

![Application Image](github-client.png)

## Building

Prerequisites
- Java JDK
- Android SDK API 25
- Gradle Build Tool

```bash
git clone https://github.com/realeigenvalue/github-client.git
cd github-client
gradle build
gradle run
```
